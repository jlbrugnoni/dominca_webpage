# Dominca Webpage

Landing page for Dominca, built with Next.js and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Main Content

- Company data, carousel copy, products, contact info, and catalog path live in `src/config/companyConfig.ts`.
- Public images live in `public/images`.
- Catalog files live in `public/catalog`.
